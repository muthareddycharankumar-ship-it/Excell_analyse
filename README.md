# Excell_analyse

A Python-based Excel analysis tool that allows you to analyse and extract insights from Excel files.

## Features
- Upload and analyse Excel files
- Extract insights from data
- Visualize data trends

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python app.py
```

## Requirements
- Python 3.x
- See `requirements.txt` for dependencies
