# 📊 Excell_analyse

A powerful web-based Excel analysis tool built for business users to upload, analyse, and extract meaningful insights from Excel files — all through a simple and intuitive interface.

---

## 🚀 Features

- 📂 Upload Excel files (`.xlsx`, `.xls`) directly through the browser
- 📊 Instant data analysis and summary statistics
- 🔍 Explore and inspect your spreadsheet data with ease
- 💼 Designed for business users — no coding required
- ⚡ Fast and lightweight Flask-based backend

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| Python | Core programming language |
| Flask | Web framework & API |
| Pandas | Data analysis & processing |
| OpenPyXL | Excel file handling |
| HTML/CSS/JS | Frontend interface |

---

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/muthareddycharankumar-ship-it/Excell_analyse.git
cd Excell_analyse
```

### 2. Create a virtual environment
```bash
python -m venv myenv
source myenv/bin/activate  # On Windows: myenv\Scripts\activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

---

## ▶️ Usage
```bash
python app.py
```

Then open your browser and go to:
```
http://localhost:5000
```

Upload your Excel file and start analysing instantly!
